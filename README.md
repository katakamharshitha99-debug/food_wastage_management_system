# Food Wastage Management System

A Streamlit-based application that helps reduce food wastage by connecting food providers and receivers.

## Features

- Dashboard KPIs
- Food Listings Management
- Provider Contact Search
- SQL Analysis
- Visualizations
- CRUD Operations
- City, Food Type, and Meal Type Filters

## Technologies

- Python
- Streamlit
- SQL Server
- Pandas
- Plotly
